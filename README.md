# 立水桥南-佳运园4区两居室找一名合租


## 位置

佳运园四区，立水桥南与立水桥之间，毗邻东小口森林公园

## 户型

90平米左右的南北通透两居室，出租的次卧大概12平米左右，主要是客厅太特么大了。。。

## 租金

押一付三，2400/月，包含物业费与取暖费，水电气平摊，宽带可以平摊，你也可以自己单拉一根。

**重点：无中介费，可以长租**

## 交通

公交107、558，直达地铁13号线 `立水桥站` 和地铁5号线 `立水桥南站` 

当然，如果你自己开车的话，小区这边的停车费用是1500/年，或150/月。小区外的路边也可以免费停，但前提是你要下班早才能占到地儿，我晚上9点下班回来基本上很难找到地儿停，晚上7点下班的时候找个停车位还是挺轻松的。

## 生活设施

附近有永辉超市、早市、新隆嘉超市、东小口森林公园，公交几站路就可以到达龙德广场、上品、卜蜂莲花，还算是方便

## 配置

公共设施：空调、洗衣机、热水器、暖气、电视机、沙发
私人的但是可以共享的设施：微波炉、烤箱、吸尘器

## 我想说的

房子已经住了3年了，好基友因为和他女友的工作地点离太远，所以搬到一个离两边都折中的地方。我个人住习惯了也不想搬家，索性就全盘接下来了，又和房东续了2年。比较佛系，能找到靠谱的室友当然最好了，找不到的话自己住一整套太奢侈了一点，但是也没办法。

如果你也喜欢清静，也喜欢户外跑步，这里绝对适合你，欢迎你来～
